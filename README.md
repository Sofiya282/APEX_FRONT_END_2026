# Apex AI Landing Page

A high-performance, ultra-responsive landing page concept built for Apex AI the next-generation AI platform designed to deploy production-ready applications in seconds. This landing page features dynamic pricing matrix controls, interactive code execute simulations, and fluid micro-interactions tailored for developer-centric SaaS products.

# Features

*   **Interactive Code Runner Simulation:** A mock IDE module allowing users to simulate executing a 3-line fast AI integration with `< 10ms` latency feedback.
*   **Dynamic Matrix-Based Pricing:** An advanced, state-isolated pricing table that dynamically recalibrates rates instantly based on:
    *   **Billing Cycle:** Toggle smoothly between Monthly and Annual plans.
    *   **Local Currency Support:** Seamless multi-currency conversion drop-down (GBP, USD, EUR, INR).
*   **Responsive Adaptive Architecture:** Features an interactive accordion framework switching layouts automatically based on user viewport context (`Map Demand`, `Route Offers`, `Track Context`).
*   **Smooth Native Motion:** High-fidelity UI transitions, entry hooks, panel accordions, and focus state isolating updates directly to the targeted text DOM nodes.
*   **Access Request Intake:** An integrated form at the footer to log interest and staging requests prior to official judge evaluation.

## 🛠️ Tech Stack

*   **Interactivity & State Management:** JavaScript
